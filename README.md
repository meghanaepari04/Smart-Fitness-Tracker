## ğŸ©º Overview
**Smart Fitness Tracker** is a desktop application built using **Java Swing**.  
It helps users calculate:
- Body Mass Index (BMI)
- Basal Metabolic Rate (BMR)
- Recommended daily calorie intake

Users can **save daily progress** and **view fitness history** anytime.  
All data is stored locally in text files (`fitness_data.txt` & `fitness_report.txt`).
